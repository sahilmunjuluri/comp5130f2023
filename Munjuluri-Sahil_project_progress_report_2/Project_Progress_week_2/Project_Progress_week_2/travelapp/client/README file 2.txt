# React Bookings Page

This React application displays and manages bookings made by a logged-in user. It utilizes React for building the user interface, Axios for making HTTP requests, and date-fns for date calculations.

This React component represents a booking widget for a place. It allows users to input their booking details, including check-in and check-out dates, number of guests, and personal information. The widget calculates the total cost and provides the option to confirm or edit the booking.


## Features

- View a list of bookings with detailed information.
- Cancel a booking with a "Cancel Booking" button.
- Display booking details such as place image, title, check-in/out dates, number of nights, number of guests, and price.
- Responsive design for a pleasant user experience.
- Input fields for check-in and check-out dates.
- Number input for the number of guests (limited by the maximum allowed guests for the place).
- Confirmation button to finalize the booking.
- Edit button to modify booking details.
- Auto-population of user details if the user is logged in.
- Calculation of the total cost based on the place's nightly rate and the number of booked nights.
- Responsive design for a seamless user experience.
